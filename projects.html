  • Digital Shuttle system

    United International University • June 2023 – Sep 2023

    Details
    GitHub Links
    • In our project, we have integrated RFID entry connected with a website, real-time
      location tracking, seat reservation, alcohol detection, and gas/smoke sensing to
      enhance safety and security measures. The RFID entry integration involves the use of
      RFID readers and tags for secure access, a microcontroller for processing RFID data,
      and the necessary wiring and connectors for hardware integration. Website
      connectivity is established through a web server for hosting the site, a database for
      storing user and sensor data, and API integration for real-time communication. Real-
      time location tracking utilizes GPS modules for precise tracking, software algorithms
      for data processing, and a user interface for location display on the website. The seat
      reservation system is based on a microcontroller for seat monitoring, a user interface
      for seat selection on the website, and a reservation algorithm for managing seat
      availability. The alcohol detection system employs alcohol sensors, a microcontroller
      for processing sensor data, and an alarm system triggered upon alcohol detection.
      Gas/smoke sensing involves sensors for detecting harmful substances, a
      microcontroller for processing data, and integration with the alarm system for
      immediate alerts. The notification system, facilitated via email, utilizes an SMTP
      server for sending notifications, programming logic for generating and sending alerts,
      and integration with the system for timely notifications. To ensure safety and security,
      the project incorporates redundancy and fail-safe mechanisms for critical components,
      regular system maintenance and updates, and user training and guidelines for system
      use.
